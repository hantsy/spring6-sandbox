# Spring 6 Sandbox

Spring 6 (and Spring Boot 3) upgrades to use Jakarta EE 9+ and Java 17 as the new code baseline. Aligned with the competitors, such as [Micronaut](https://micronaut.io/), [Quarkus](https://quarkus.io/), Spring 6 adds built-in native building support which was [an experimental project](https://github.com/spring-projects-experimental/spring-native/releases/tag/0.11.0). At the same time, a lot of long-deprecated features are removed in Spring 6.0.
